```bash

npm i -g openclaw@latest
# 使用openclaw升级
openclaw update
```

### 问题7：如何登录ClawHub并设置Token？

ClawHub 是 **OpenClaw 的公共 Skills 注册中心**。它是一项免费服务：所有 Skills 都是公开的、开放的，所有人都可以查看、共享和复用。Skills 就是一个包含 `SKILL.md`​ 文件（以及辅助文本文件）的文件夹。你可以在网页应用中浏览 Skills，也可以使用 CLI 来搜索、安装、更新和发布 Skills。网站：[clawhub.com](https://clawhub.com/)

- 登录[ClawHub网站](https://clawhub.ai/settings)，`Create token`，然后复制token
- 在命令行或Docker `/bin/bash`​ 中输入：`clawhub login --token <token>`

  ```bash
  # 安装clawhub
  npm i -g clawhub
  # 认证
  clawhub login #（浏览器流程）
  clawhub login --token <token> # 直接输入token，推荐
  # 查看安装了哪些技能
  clawhub list
  # 安装技能
  clawhub install my-skill-pack
  # 更新
  clawhub update --all
  ```
- 具体详见[https://docs.openclaw.ai/zh-CN/tools/clawhub](https://docs.openclaw.ai/zh-CN/tools/clawhub)
